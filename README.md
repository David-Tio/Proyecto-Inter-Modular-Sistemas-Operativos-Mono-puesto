## Descripción
 
Este proyecto consiste en la instalación, configuración y gestión de un sistema operativo en una máquina virtual, simulando el puesto de trabajo tipo de un empleado de la Inmobiliaria Chamartín.
 
---
 
## Sistema operativo utilizado
 
El sistema operativo instalado y configurado en este proyecto es Windows 11 Pro, instalado sobre una máquina virtual en VirtualBox, simulando el puesto de trabajo de un empleado de la Inmobiliaria Chamartín.

## Resumen del proceso
 
### 1. Instalación del SO
- Descarga del fichero `.ISO` desde la web oficial de Microsoft
- Creación de la máquina virtual en VirtualBox (8 GB RAM, 2 CPUs)
- Instalación de Windows 11 Pro sin clave de producto
### 2. Configuración básica
- Nombre del equipo: `PC-AGENTE01`
- Idioma y región: Español (España)
- Actualizaciones del sistema aplicadas
### 3. Gestión de usuarios
- `Producción01` – Usuario estándar (empleado)
- `Admin` – Usuario administrador (IT)
- Usuarios creados mediante `start ms-cxh://localonly` (sin cuenta Microsoft)
### 4. Software instalado
- **Microsoft Office 365** – Suite ofimática
- **AVG Antivirus Free** – Protección del sistema
- **Brave** – Navegador web seguro
### 5. Medidas de seguridad
- Firewall de Windows Defender activado
- Windows Update al día
- Escaneo de vulnerabilidades con AVG
